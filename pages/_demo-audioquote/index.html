Audioquote

Un témoignage audio sous-titré, qui peut se déclencher automatiquement à l'entrée dans l'écran.
http://localhost:3000/audio/do-maj.mp3 http://localhost:3000/audio/do-maj.srt 3,6,8 La gamme de Do majeur : Jouer Arrêter Activer le son Couper le son false true true false false
audioSrc?: string L'URL du fichier audio à jouer
subsSrc?: string L'URL du fichier sous-titres (.srt)
subsGroups?: string|Array<string|number> Permet de grouper des lignes de sous-titres ensemble. Il suffit de préciser l'ID de la dernière ligne de sous-titres pour chaque groupe. Exemple : si l'on veut grouper les sous-titres 1 à 7, 8 à 12, 13 à 19 et enfin 20 jusqu'à la fin, on donnera à cette propriété l'une de ces valeurs : 7, 12, 19 (le dernier groupe est déduit).
title?: string|html Un titre à afficher dans l'app
playButton?: string|html Le contenu à afficher dans le bouton play
pauseButton?: string|html Le contenu à afficher dans le bouton pause
loudButton?: string|html Le contenu à afficher dans le bouton loud
muteButton?: string|html Le contenu à afficher dans le bouton mute
hidePauseButton?: boolean Permet de masquer le bouton pause
autoPlayWhenVisible?: boolean Déclenche la lecture des sous-titres quand l'app entre dans l'écran (si la lecture n'a pas été terminée déjà une fois et si l'utilisateur n'a pas manuellement mis la lecture en pause.)
autoPauseWhenHidden?: boolean Force la mise en pause de la lecture si l'app sort de l'écran.
autoLoudWhenVisible?: boolean Déconseillé, et ne fonctionne que si l'utilisateur a déjà cliqué quelque part dans la page. Active le son de l'audio lorsque l'app rentre dans l'écran.
autoMuteWhenHidden?: boolean Désactive le son de l'audio si l'app sort de l'écran.
